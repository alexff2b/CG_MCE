# CG_MCE

## Overview

CG_MCE (Classic Game / Computer Modular Emulator)

A modular emulator framework written in C.

Current CPU:
- Z80

Goals:
- Accurate hardware emulation
- Modular architecture
- Easy addition of new CPUs
- Terminal debugger
- Retro computer research
